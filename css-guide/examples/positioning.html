Exemplos de Posicionamento em CSS

Posicionamento Estático

Este é um exemplo de posicionamento estático. Os elementos são posicionados de acordo com o fluxo normal do documento.

Posicionamento Relativo

Este é um exemplo de posicionamento relativo. O elemento é posicionado em relação à sua posição original.

Posicionamento Absoluto

Este é um exemplo de posicionamento absoluto. O elemento é posicionado em relação ao seu ancestral mais próximo que não tenha um posicionamento estático.

Posicionamento Fixo

Este é um exemplo de posicionamento fixo. O elemento é posicionado em relação à janela de visualização e permanece fixo durante o scroll.

Posicionamento Sticky

Este é um exemplo de posicionamento sticky. O elemento se comporta como relativo até que seu bloco pai seja rolado para fora da visualização.
