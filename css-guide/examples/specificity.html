Especificidade em CSS

Entenda como a especificidade afeta a aplicação de estilos em seus elementos.

O que é Especificidade?

A especificidade é um mecanismo que o navegador usa para determinar qual regra CSS deve ser aplicada a um elemento quando várias regras podem se aplicar.

Como Funciona?

A especificidade é calculada com base em três categorias:

  • Seletores de ID (mais específicos)
  • Seletores de classe, atributos e pseudo-classes
  • Seletores de elementos e pseudo-elementos (menos específicos)
