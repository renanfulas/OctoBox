# CSS Guide

Este guia de CSS fornece uma estrutura abrangente para o desenvolvimento de estilos consistentes e eficientes. Ele é dividido em várias seções, cada uma abordando diferentes aspectos do CSS, desde a base até componentes e utilitários.

## Estrutura do Projeto

O projeto é organizado da seguinte forma:

- **src/styles**: Contém todos os arquivos CSS.
  - **base**: Estilos básicos que garantem consistência.
    - `reset.css`: Redefinição de estilos para todos os navegadores.
    - `typography.css`: Estilos de tipografia.
    - `variables.css`: Variáveis CSS reutilizáveis.
  - **components**: Estilos para componentes reutilizáveis.
    - `buttons.css`: Estilos para botões.
    - `cards.css`: Estilos para cartões.
    - `forms.css`: Estilos para formulários.
    - `modals.css`: Estilos para modais.
    - `navigation.css`: Estilos para navegação.
  - **layout**: Estilos para layout.
    - `grid.css`: Sistema de grid.
    - `flexbox.css`: Layout flexível.
    - `responsive.css`: Estilos responsivos.
  - **utilities**: Classes utilitárias.
    - `spacing.css`: Classes para espaçamento.
    - `colors.css`: Classes para cores.
    - `animations.css`: Classes para animações.
  - `main.css`: Ponto de entrada principal para os estilos.

- **src/pages**: Contém as páginas HTML do guia.
  - `index.html`: Página inicial.
  - `typography.html`: Informações sobre tipografia.
  - `colors.html`: Paleta de cores.
  - `components.html`: Lista de componentes.
  - `layout.html`: Informações sobre layout.
  - `animations.html`: Exemplos de animações.
  - `responsive.html`: Técnicas para layouts responsivos.

- **src/assets/fonts**: Estilos para fontes personalizadas.
  - `fonts.css`: Estilos de fontes.

- **docs**: Documentação adicional.
  - `principles.md`: Princípios do design CSS.
  - `naming-conventions.md`: Diretrizes de nomenclatura.
  - `best-practices.md`: Melhores práticas de CSS.
  - `accessibility.md`: Acessibilidade no design CSS.
  - `performance.md`: Dicas de otimização de desempenho.

- **examples**: Exemplos práticos de CSS.
  - `selectors.html`: Exemplos de seletores.
  - `specificity.html`: Discussão sobre especificidade.
  - `box-model.html`: Explicação do modelo de caixa.
  - `positioning.html`: Exemplos de posicionamento.
  - `media-queries.html`: Exemplos de consultas de mídia.

## Como Usar

Para começar a usar este guia, você pode abrir a página `index.html` no seu navegador. A partir daí, você pode navegar pelas diferentes seções para aprender sobre tipografia, cores, componentes e muito mais.
