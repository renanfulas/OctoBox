# Aesthetic Directions

Use this file when the interface needs a fast but intentional visual point of view.

## Direction Menu

### Editorial Command

Feel:

- magazine-grade hierarchy
- dramatic type contrast
- structured asymmetry

Signals:

- refined serif display with restrained sans body
- large headlines, tight subheads, generous whitespace
- sharp black, bone, ink, burgundy, or brass accents

Memorable move:

- oversized masthead-like headline paired with disciplined information rails

### Mineral Futurism

Feel:

- premium-tech
- glass, vapor, signal light
- near-future control room

Signals:

- distinctive display sans or techno serif with clean body companion
- dark mineral surfaces, cyan or ice-blue guidance, controlled magenta accents
- soft atmospheric gradients and short glows

Memorable move:

- luminous focal card emerging from a quiet, deep background

### Playful Toy-System

Feel:

- tactile
- cheerful
- smart rather than childish

Signals:

- chunky display font with cleaner supporting text
- bold primary blocks, rounded forms, stacked cards, playful motion
- candy, enamel, or classroom-inspired palette

Memorable move:

- interface pieces that feel like physical toys snapping together

### Brutalist Utility

Feel:

- raw
- direct
- unapologetically functional

Signals:

- monospaced or sharply geometric display pairing
- hard borders, loud contrast, exposed grid logic
- black, off-white, safety orange, electric blue, or acid accents

Memorable move:

- overscaled labels and rigid structure used as the aesthetic itself

### Organic Precision

Feel:

- natural
- calm
- premium without coldness

Signals:

- humanist serif or calligraphic display with warm body text
- moss, clay, sand, smoke, stone palettes
- textured backgrounds, soft contour shadows, flowing spacing

Memorable move:

- a precise interface wrapped in materials that feel hand-touched

### Retro-Future Poster

Feel:

- cinematic
- collectible
- high-drama

Signals:

- expressive poster-style display face plus narrow technical body font
- dramatic gradients, halftone or grain textures, radial lighting
- strong diagonals and layered overlays

Memorable move:

- a hero section that feels like a festival poster you can click through

## Typography Rules

- Pair one voice font with one support font.
- Let the display face carry personality; let the body face carry endurance.
- Adjust tracking, weight, casing, and line length to match the direction.
- Avoid the same font pair across different builds.

## Color Rules

- Pick a dominant base and 1 or 2 sharp accents.
- Reserve the most saturated color for decisions, focus, or signature.
- Keep semantic status colors distinct from brand accents.
- Build atmosphere with layered surfaces, not random gradients.

## Motion Rules

- Choose one hero moment: load-in, reveal, hover lift, scroll trigger, or panel choreography.
- Staggered entrances usually beat many unrelated micro-interactions.
- Use motion to teach hierarchy and cause-effect.
- If every element moves, nothing feels important.

## Layout Rules

- Break symmetry on purpose.
- Use overlap, rails, off-grid moments, or silhouette changes when they strengthen the concept.
- Match density to mood: editorial calm needs air; maximalist energy needs structure.
- Make mobile feel like a re-composed poster, not a crushed desktop.

## Anti-Slop Checklist

Before shipping, ask:

- Could this be mistaken for a random template marketplace demo?
- Did I choose this font because it is right, or because it is common?
- Is the palette memorable after five seconds?
- Is there one visual decision people will remember?
- Does the interface still work if the decorative layer is removed?

If the answer is weak, the direction is not committed enough yet.
