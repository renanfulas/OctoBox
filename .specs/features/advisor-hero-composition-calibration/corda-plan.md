# Corda Plan

## Executive North

The advisor hero is already strong.
This calibration is not about redesigning it.

It is about deciding whether the new centered composition improves:

1. first reading
2. command presence
3. emotional tone

without weakening:

1. urgency
2. hierarchy
3. operational credibility

## Risk

If we over-center the hero, it may become too editorial and lose dashboard gravity.

## Strategy

1. inspect the local composition change
2. compare it against the live hero contract
3. decide keep / soften / discard
