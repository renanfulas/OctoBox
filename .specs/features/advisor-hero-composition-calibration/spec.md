# Specification

## Problem

A local dashboard change currently centers the advisor hero composition more strongly, but it has not yet been validated against the `Front Display Wall`.

## Goal

Decide whether the centered hero treatment should:

1. ship as-is
2. ship in a softer form
3. be rejected

## Non-goals

1. rewriting dashboard copy
2. changing payload or priority semantics
3. redesigning the full advisor dashboard

## Success criteria

This package succeeds when:

1. the hero composition decision is explicit
2. the resulting hero remains readable and authoritative
3. the change can be committed without ambiguity
