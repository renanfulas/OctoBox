# Corda Plan

## Executive North

The dashboard pilot succeeded because it improved atmosphere and product presence without lying about urgency or next action.

This consolidation phase must protect that gain.

## Core question

What from the dashboard premium pilot becomes official OctoBox grammar, and what should stay local to the dashboard?

## Risks

1. exporting exact glow values instead of exporting principles
2. spreading dashboard drama into operational screens that need quieter reading
3. confusing "premium" with "more effects"

## Strategy

1. inspect pilot outputs by category
2. classify them into export / adapt / keep-local
3. define the first non-dashboard target only after that grammar is explicit
