# Context

The dashboard premium pilot proved that OctoBox can absorb a more premium visual language without violating the `Front Display Wall`.

That success creates a new risk:

1. copying the dashboard skin everywhere
2. spreading glow and gradient values without semantic discipline
3. turning a good pilot into a new visual mosaic

This package exists to prevent that.

The next move is not "make more pages look like the dashboard."

The next move is:

1. extract the grammar
2. define export rules
3. preserve local exceptions where they belong

In simple language:

We already proved the suit looks good on the mannequin.
Now we need to decide which cuts become official tailoring rules for the whole wardrobe.
