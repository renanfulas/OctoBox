# Context

Dashboard proved the premium grammar. Finance proved that the grammar can expand into another command surface without losing transactional clarity.

Reception is the next strategic surface, but it is different by nature:

1. it is faster
2. it is more human-facing
3. it behaves like a counter, not a control room

That means the expansion must be quieter, clearer, and lighter than dashboard or finance while still feeling like the same product family.
