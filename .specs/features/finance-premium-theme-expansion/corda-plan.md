# Corda Plan

## Executive North

Finance is the first expansion because it behaves like a decision center and can absorb richer premium grammar with lower risk than reception or student form.

## Core question

How do we translate premium dashboard grammar into finance without weakening transactional clarity?

## Risks

1. importing dashboard glow intensity instead of finance-appropriate calm
2. making queue, movements, and portfolio look equally dramatic
3. confusing premium atmosphere with visual noise in a transactional screen

## Strategy

1. map finance command and support surfaces
2. apply only exportable grammar plus adapted support depth
3. validate that command order stays clearer, not prettier-only
