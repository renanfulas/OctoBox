# Context

The dashboard premium pilot proved the visual language.

The consolidation phase proved that finance is the safest first non-dashboard target.

That does not mean finance should become "the dashboard in another costume."

It means finance should inherit:

1. premium shell coherence
2. layered support surfaces
3. calmer premium borders and shadows
4. semantic intensity that still obeys operational truth

In simple language:

We are not copying the stage spotlight into the billing room.
We are upgrading the billing room with the same architectural language, but quieter and more precise.
