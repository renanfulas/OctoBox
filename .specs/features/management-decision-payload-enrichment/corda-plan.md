# CORDA Plan

## North

Enrich the strategic payloads so the front stops guessing which signal matters most.

## Why now

Recent work improved:

1. structure
2. route contracts
3. reading order
4. flow honesty

That means the next strongest ROI is now behind the template layer.

## Risk

This is a medium-risk strategic pass.

Why:

1. payload changes can affect multiple surfaces at once
2. a bad enrichment can add noise instead of clarity
3. too much prose in snapshots becomes cognitive debt

## Guardrails

1. prefer small, sharp fields over long narrative blobs
2. enrich priority without duplicating template copy unnecessarily
3. keep query cost visible while improving semantic value
4. let presenters format the facade, but let snapshots carry real reasoning

## Execution waves

1. map live management payloads and identify flat signals
2. enrich one surface at a time, starting with highest operational leverage
3. validate that the UI becomes easier to decide, not just more verbose
