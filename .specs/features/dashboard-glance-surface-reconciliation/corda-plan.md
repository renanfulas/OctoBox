# CORDA Plan

## North

Reconcile the dashboard glance strip so its visual intensity, severity classes, and actionable state tell one coherent story.

## Why now

This is the next strategic pass because:

1. the dashboard is a control surface, not a peripheral page
2. glance cards are among the most visible urgency indicators in the product
3. the remaining uncertainty is more semantic than structural

## Risks

The biggest risk is not breakage. It is signal confusion:

1. too much neon can turn urgency into wallpaper
2. too little distinction can flatten emergency, urgent, and risk into the same emotion
3. actionability and severity can drift apart if the classes tell different stories

## Execution sequence

1. map the live severity contract
2. inspect whether the neon layer is canonical or overreaching
3. reconcile the surface with minimal, intentional changes
4. validate that the strip stays readable and decisive

## Success bar

This pass is done when:

1. glance severity and actionability speak one visual language
2. the neon layer feels intentional rather than ornamental
3. the dashboard keeps high urgency clarity without turning noisy
