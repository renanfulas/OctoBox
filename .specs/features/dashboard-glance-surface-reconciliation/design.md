# Design

## Canonical rule

The glance strip should communicate three layers clearly:

1. severity
2. actionability
3. next attention target

## Surface split

The template should own:

1. semantic class composition
2. content and routing

The CSS should own:

1. visual intensity
2. pulse behavior
3. border and indicator harmonization

## Reconciliation lens

We should judge the surface by one question:

Does the strip feel like a control panel with disciplined lights, or like a showcase where every lamp is trying to scream?

## Residual tolerance

If the current neon layer is already coherent, the right outcome may be documentation and validation rather than heavy edits.
