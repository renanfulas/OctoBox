# Trilha de leitura para Dev Senior

## Objetivo

Esta trilha existe para quem vai:

1. tomar decisao estrutural
2. revisar acoplamento
3. abrir corredor novo
4. arbitrar trade-off tecnico

## O que esse perfil deve observar

1. o que ja e corredor oficial
2. onde ainda existe transicao
3. onde uma nova abstracao reduziria acoplamento real
4. onde uma nova abstracao seria so teatro arquitetural

## Regra curta

Senior no OctoBox nao deve perguntar apenas "onde eu mexo?".

Deve perguntar:

1. qual corredor essa mudanca fortalece?
2. qual bypass ela elimina?
3. qual custo cognitivo ela reduz para o time?
